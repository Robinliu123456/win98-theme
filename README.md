## 介绍
基于perfree: [http://perfree.org.cn](http://perfree.org.cn)
基于win98: [https://github.com/1j01/98](https://github.com/1j01/98)
开发的win98怀旧主题

## 使用
根据perfree官网部署项目, 下载对应版本的Releases版本包, 安装主题即可
